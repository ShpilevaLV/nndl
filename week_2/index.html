8 Feature Importance

Train model to analyze feature importance.

üìä How it works: Feature importance is calculated using permutation importance. Each feature is shuffled, and we measure how much the model's accuracy decreases. Greater decrease = more important feature.
  • Positive importance: Feature helps predict survival
  • Negative importance: Feature helps predict non-survival
  • Near zero: Feature has little impact

üöÄ Deployment Notes

How to deploy this application:

  1. Create a public GitHub repository
  2. Commit index.html and app.js to the repository
  3. Go to repository Settings ‚Üí Pages
  4. Select "Deploy from branch" and choose "main" branch and "/ (root)" folder
  5. Click "Save" - your app will be available at:
    https://[your-username].github.io/[repository-name]
  6. Wait a few minutes for GitHub Pages to build and deploy

To use with other datasets:

Modify the schema configuration in app.js:

  • TARGET_FEATURE: Your binary classification target column
  • ID_FEATURE: Identifier column to exclude
  • NUMERICAL_FEATURES: List of numerical feature columns
  • CATEGORICAL_FEATURES: List of categorical feature columns
